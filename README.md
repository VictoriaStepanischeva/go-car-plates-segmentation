go get github.com/VictoriaStepanischeva/go-car-plates-segmentation

### Usage
```bash
go run main.go -h
Usage of main:
	-dstd string
		directory with destination images (default "/tmp")
	-pattern string
		pattern for source files (default "*")
	-srcd string
		directory with source images (default ".")
```
